 TrendCart Analytics Dashboard
  Overview

The **TrendCart Analytics Dashboard** is an interactive data visualization project built in **Microsoft Excel** to analyze e-commerce performance. It provides insights into customer behavior, product trends, sales performance, and operational efficiency.

---

Features

*  **Sales Trend Analysis** (Quantity vs Amount over time)
*  **Customer Purchase Behavior**
*  **Customer Satisfaction Analysis**
*  **Order Frequency Distribution**
*  **Product Popularity Insights**
*  **Geographical Sales Distribution**
*  **Shipping Time Analysis**
*  **Interactive Filters (Gender, Order Source)**

---

 Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning & Transformation
* Dashboard Design

---

 Dataset

* Contains 2000+ records of customer transactions
* Includes:
  * Product details
  * Customer demographics
  * Order source (App, Website, Instagram, etc.)
  * Ratings & delivery metrics

 Key Insights

* Majority of customers prefer **App & Website purchases**
* **Female customers dominate (52%)** purchase share
* Certain products like **T-shirts and Jeans** are top-selling
* Delivery time mostly falls between **1–3 days**
* Sales are concentrated in specific regions

---

 Use Cases

* Business performance monitoring
* Customer segmentation
* Sales optimization strategies
* Operational efficiency improvement

---

 How to Use

1. Download the Excel file
2. Open in Microsoft Excel (preferably latest version)
3. Use filters (Gender, Order Source) to explore insights
4. Interact with charts for dynamic analysis
